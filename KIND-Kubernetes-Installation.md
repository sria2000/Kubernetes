# KIND (Kubernetes IN Docker) Installation and Cluster Setup

## Overview

Kind (Kubernetes IN Docker) is a tool used to run Kubernetes clusters inside Docker containers.

It is mainly used for:

- Kubernetes learning
- Testing applications
- Local development
- Testing multi-node Kubernetes scenarios
